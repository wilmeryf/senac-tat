# Atualização de Drivers e Configuração do Sistema  

> **Data:** 02 de fevereiro de 2026  

Após a instalação do sistema operacional, foi realizada a atualização dos drivers para garantir estabilidade, compatibilidade e melhor desempenho do computador.

---

## Atualização pelo Windows Update  

Foi utilizado o Windows Update para:

- Buscar atualizações do sistema  
- Instalar drivers reconhecidos automaticamente  
- Atualizar componentes como rede, vídeo e chipset  

---

## Atualização com Software Auxiliar  

Também foi utilizado o Driver Booster para identificar e atualizar drivers que não foram instalados automaticamente pelo Windows.

---

## Verificação Final  

Após as atualizações, foi feita a conferência no Gerenciador de Dispositivos para garantir que não havia erros ou dispositivos sem driver.

---

## Resultado  

O sistema ficou atualizado, com todos os drivers instalados corretamente e pronto para testes de desempenho.
