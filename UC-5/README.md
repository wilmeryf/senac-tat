# UC-5

Este diretório reúne todas as atividades práticas realizadas na UC-5, com foco na apresentação de redes locais, instalação de equipamentos úteis, integração de tecnologia na rede voltada para IoT.

## Conteúdo

- Apresentação de Redes Locais
- Configuração de Roteadores sem fio
- Cisco Packet Tracer
- Montagem de uma rede doméstica
- Arduino com Ethernet Shield
- Implementação do arduino na rede voltada para IoT

Atividades realizadas no fim de fevereiro e grande parte de março de 2026 como parte das aulas práticas da UC-5.
